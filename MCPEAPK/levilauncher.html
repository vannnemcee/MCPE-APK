Apa itu LeviLauncher


LeviLauncher adalah launcher mc open source yang dirancang khusus untuk para pemain Minecraft Bedrock Edition untuk memuat dan mengelola mod dengan mudah serta menyediakan API yang kaya untuk fungsi yang ditingkatkan. Peluncur ini memungkinkan pengguna untuk menambahkan dan mengatur berbagai mod, mengimpor paket resource atau shader seperti mc patch.
